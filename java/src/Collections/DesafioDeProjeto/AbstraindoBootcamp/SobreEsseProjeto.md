# Projeto - Abstraindo um Bootacamp

## Esse projeto foi produzido seguindo a matéria "abstraindo um bootcamp" do módulo **Estrutura de dados e API** de Estreams do curso de Java Developer, disponibilizado na plataforma da <img src="https://hermes.digitalinnovation.one/assets/diome/logo-minimized.png" alt="Logo da DIO" width="20"/> [DIO](https://web.dio.me/track/formacao-java-developer).

- POO
- Streams API
- Collections
